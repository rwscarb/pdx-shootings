# [pdx-shootings](https://map.pdx-shootings.com/)

## Project Setup

```sh
npm i
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Fetch the latest data

```sh
$ ./scripts/fetch-shootings.py
$ ./scripts/fetch-barrels.py
```

### Deploy (make sure to fetch first)

```sh
$ ./scripts/deploy.sh
```
